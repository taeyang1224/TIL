# Kream 인기상품 크롤링
- selenium을 활용하여 동적페이지 크롤링
- kream 랭킹을 크롤링하여 csv로 저장
- 단순 개인 프로젝트로 활용할 것으로 크롤링 함

## kream.csv 
### Coulmn 설명

- brand: 브랜드명
- en_name: 상품 영문
- kor_name: 상품 한글
- product_num: 상품 번호
- model: 모델명
- release: 발매일
- color: 색상
- price: 발매 가격
- re_price: 리셀 가격
- wish: 위시 수
- review: 후기 수
- link: 상품 링크
